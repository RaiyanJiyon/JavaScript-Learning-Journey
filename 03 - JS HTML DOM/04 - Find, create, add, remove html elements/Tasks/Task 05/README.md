### **Task 5: Dynamically Add and Remove Elements**
1. Create a to-do list where users can add and remove items.

**Requirements:**
- Create an input field and a button labeled "Add Task".
- When the button is clicked, the value from the input field should be added as a new list item to the to-do list.
- Each item should have a "Remove" button next to it that, when clicked, removes the item from the list.
